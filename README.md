## 💻 How to Run

1. Clone the repository:
```bash
git clone https://github.com/VedK5643/DSA-CPP.git
```
2.Navigate to the folder and compile:
```
g++ filename.cpp -o filename
```
3.Run the executable:
```
./filename
```
